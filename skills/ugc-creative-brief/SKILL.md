# UGC Creative Brief Generator

Generates complete user-generated content (UGC) creator briefs including hook formulas, shot list templates, platform-specific format requirements, CTA placement strategies, AI UGC tool recommendations, and creator compensation frameworks.

## Capabilities

- **Creator Brief Generation**: Full brief documents with brand context, deliverables, dos/don'ts
- **Hook Formula Library**: Problem-agitation-solution, before-after, testimonial, curiosity gap
- **Platform Format Specs**: TikTok 9:16, Meta 1:1 & 4:5, YouTube 16:9, Pinterest 2:3
- **Shot List Templates**: Structured shot-by-shot breakdowns with timing
- **CTA Placement Strategy**: When and how to place calls-to-action for max conversion
- **AI UGC Tool Comparison**: Arcads, Creatify, HeyGen, Synthesia, D-ID feature comparison
- **Creator Compensation**: Rate cards, usage rights pricing, performance bonuses

## Workflows

### 1. Creator Brief Generation

**Required inputs:**
- Brand name and product/service
- Target audience demographics and psychographics
- Campaign objective (awareness, consideration, conversion)
- Platform(s) for distribution
- Budget range
- Key messages and talking points
- Any restricted claims or compliance requirements

**Brief template:**

```markdown
# UGC Creator Brief

## Brand Overview
- **Brand:** [Brand Name]
- **Product:** [Product/Service]
- **Website:** [URL]
- **Brand Voice:** [Casual/Professional/Edgy/Warm]

## Campaign Details
- **Objective:** [Awareness / Consideration / Conversion]
- **Platform(s):** [TikTok, Instagram Reels, YouTube Shorts, Meta Feed]
- **Target Audience:** [Age, gender, interests, pain points]
- **Campaign Duration:** [Start - End]

## Deliverables
- [ ] X videos at [length] seconds
- [ ] Format: [9:16 / 1:1 / 4:5]
- [ ] Resolution: 1080x1920 minimum
- [ ] Raw footage delivery: [Yes/No]
- [ ] Revision rounds: [1-2]

## Key Messages (Pick 2-3 per video)
1. [Primary value proposition]
2. [Differentiator / unique feature]
3. [Social proof / result]

## Hook Requirements
- First 3 seconds MUST stop the scroll
- Use one of the approved hook formulas (see below)
- Do NOT start with brand name

## CTA Requirements
- Verbal CTA at [timestamp]
- On-screen CTA text: "[Specific CTA]"
- Promo code: [CODE] (if applicable)

## DO's
- ✅ Film in natural lighting
- ✅ Use personal phone (authentic feel)
- ✅ Show genuine reactions
- ✅ Include product in frame within first 5 seconds
- ✅ Speak directly to camera

## DON'Ts
- ❌ No competitor mentions
- ❌ No unsubstantiated health/income claims
- ❌ No background music with copyright issues
- ❌ No filters that obscure the product
- ❌ No scripted-sounding delivery

## Shipping / Product Access
- Product will be shipped to your address by [date]
- OR access credentials will be sent to your email

## Timeline
- Brief acceptance: [Date]
- Content draft: [Date + 5 days]
- Revisions: [Date + 7 days]
- Final delivery: [Date + 10 days]

## Compensation
- Base fee: $[amount]
- Usage rights: $[amount] per platform per [30/60/90] days
- Performance bonus: $[amount] per [metric threshold]
- Payment terms: Net [15/30] via [PayPal/Wise/direct]
```

### 2. Hook Formulas

#### Problem-Agitation-Solution (PAS)

```
[0-3s] HOOK: "Stop wasting money on [pain point]"
[3-8s] AGITATE: "I used to [describe frustrating experience]..."
[8-20s] SOLUTION: "Then I found [Product] and everything changed"
[20-25s] PROOF: Show result / transformation
[25-30s] CTA: "Link in bio to try it yourself"
```

**Best for:** Products solving clear pain points, B2C, health/wellness, productivity tools

#### Before-After-Bridge (BAB)

```
[0-3s] HOOK: Show the "before" state (messy desk, bad skin, slow workflow)
[3-8s] BEFORE: "This was me [timeframe] ago..."
[8-12s] BRIDGE: "I started using [Product]..."
[12-25s] AFTER: Show transformation with genuine excitement
[25-30s] CTA: "Get [X]% off with my code [CODE]"
```

**Best for:** Visual transformations, beauty, home, fitness, SaaS tools

#### Testimonial / Story

```
[0-3s] HOOK: "I never thought I'd say this but..."
[3-10s] CONTEXT: Establish credibility and relatability
[10-20s] EXPERIENCE: Describe specific use case and results
[20-25s] ENDORSEMENT: "I genuinely recommend this to anyone who..."
[25-30s] CTA: "Check it out, link in bio"
```

**Best for:** High-trust products, subscriptions, services, B2B

#### Curiosity Gap / Pattern Interrupt

```
[0-3s] HOOK: Unexpected visual or statement that creates curiosity
        Examples: "POV: you discover [category] actually works"
                  "The [product type] that broke the internet"
                  Start mid-action with no context
[3-8s] BUILD: Tease the reveal without giving it away
[8-20s] REVEAL: Show the product and explain
[20-30s] CTA: "Trust me, you need this"
```

**Best for:** TikTok-native content, Gen Z audiences, viral potential

#### Listicle / Tips Format

```
[0-3s] HOOK: "[Number] things I wish I knew about [topic]"
[3-8s] TIP 1: Quick actionable tip (product tie-in)
[8-15s] TIP 2: Another tip with product demo
[15-22s] TIP 3: Most impactful tip with proof
[22-30s] CTA: "Follow for more + link in bio"
```

**Best for:** Educational content, thought leadership, multi-feature products

| 6 | 35-40s | Talking head | Personal endorsement | "Game changer for me" |
| 7 | 40-45s | CTA card | Promo code on screen | "Use code [X] for [Y]% off" |

#### Testimonial / Day-in-Life (60 seconds)

| Shot # | Duration | Shot Type | Description | Audio |
|--------|----------|-----------|-------------|-------|

| 2 | 3-10s | Talking head | Introduce self + context | "As a [profession/role]..." |
| 3 | 10-18s | B-roll + VO | Show daily routine WITH product | Voiceover: how product fits |
| 4 | 18-30s | Demo | Product in action, key feature | Narrate specific benefit |
| 5 | 30-42s | Talking head | Share specific results | "In [timeframe], I [result]" |
| 6 | 42-50s | B-roll | Lifestyle shots with product | Aspirational feel |

### 5. CTA Placement Strategies

| Strategy | Placement | Best For | Conversion Rate |
|----------|-----------|----------|-----------------|
| End Card | Last 3-5 seconds | Awareness campaigns | 1-3% CTR |
| Mid-Roll + End | At transition + end | Consideration | 3-5% CTR |
| Native Mention | Woven into narrative | High-trust products | 5-8% CTR |
| Pinned Comment | First comment | TikTok/IG organic | 2-4% CTR |
| Bio Link | "Link in bio" verbal | Organic UGC | 1-2% CTR |
| Swipe-Up/Link Sticker | Platform CTA | Paid amplification | 4-7% CTR |

**CTA copy formulas:**

```
Urgency: "Only [X] left at this price — link in bio"
Social Proof: "Join [X,000] people who already [result]"
Risk Reversal: "Try it free for 30 days, link below"
Exclusive: "Use my code [CODE] for [X]% off — only for my followers"
Curiosity: "I put the link in my bio so you can see for yourself"
```

### 6. AI UGC Tools Comparison

| Feature | Arcads | Creatify | HeyGen | Synthesia | D-ID |
|---------|--------|----------|--------|-----------|------|
| **Pricing** | $100/mo (20 videos) | $49/mo (5 videos) | $59/mo (3 min) | $29/mo (10 min) | $5.90/min |

| **Lip Sync** | Excellent | Good | Excellent | Good | Good |
| **Script Input** | URL or text | URL or text | Text | Text | Text |
| **Batch Create** | Yes (bulk) | Yes | No | Yes | Yes |
| **Ad Platform Export** | Direct | Direct | Manual | Manual | Manual |
| **B-Roll Support** | Yes | Yes | Limited | No | No |
| **Hook Variations** | Auto-generate | Auto-generate | Manual | Manual | Manual |
| **Best For** | Paid ad UGC at scale | Quick ad tests | Training/corp | Enterprise | API integration |

**Recommendation by use case:**

```
IF objective == "performance_ads" AND budget > $500/mo:
  USE Arcads (best ad-specific features, bulk creation)

IF objective == "testing_hooks" AND budget < $200/mo:
  USE Creatify (cheapest per video, URL-to-ad)

IF objective == "corporate_training" OR languages > 5:
  USE Synthesia (most languages, professional tone)

IF objective == "personalized_outreach":
  USE HeyGen (best lip sync, custom avatars)

IF objective == "api_integration":
  USE D-ID (best API, pay-per-minute)
```

#### Usage Rights Pricing

| License Type | Multiplier | Description |
|-------------|------------|-------------|
| Organic only | 1x base | Post on creator's channel only |
| Paid amplification | 1.5-2x base | Brand runs as paid ad |
| Whitelisting | 2-3x base | Brand runs from creator's handle |
| Perpetual | 3-5x base | Unlimited use forever |
| Exclusivity | +50-100% | Cannot work with competitors for X months |

#### Performance Bonus Structure

```
Tier 1: $X per 1,000 views over baseline
Tier 2: $X per conversion attributed
Tier 3: $X if video exceeds [CTR/engagement] threshold

Example for D2C brand:
  Base: $200 for 1 TikTok + 1 Reel
  Usage: +$150 for 60-day paid amplification
  Bonus: +$50 per 100K views over 50K
  Bonus: +$5 per verified purchase (tracked via unique code)
  Total potential: $200-800 per creator
```

## Examples

### Example: SaaS Product UGC Brief

```markdown
# UGC Brief: ProjectFlow (Project Management SaaS)

## Hook: Problem-Agitation-Solution
"Is anyone else drowning in Slack messages and missed deadlines?"
→ Show chaotic workflow (multiple tabs, notifications)
→ "I switched to ProjectFlow and honestly? Life changing."
→ Screen recording of clean dashboard
→ "Free trial link in bio — thank me later"

## Deliverables
- 2x TikTok/Reels (9:16, 30 seconds each)
- 1x Meta Feed (1:1, 30 seconds)
- Raw footage of all takes

## Shot List
1. [0-3s] Talking head, frustrated expression, "Okay I need to talk about this"
2. [3-8s] Screen recording: messy inbox, notifications, chaos
3. [8-12s] Back to camera: "My coworker showed me ProjectFlow..."
4. [12-22s] Screen demo: create project, drag tasks, clean UI
5. [22-27s] Talking head: genuine amazement, "Everything in one place"
6. [27-30s] CTA overlay: "Free 14-day trial ↓"
```
